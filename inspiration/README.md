# inspiration

Reference artwork organised by painting. One subfolder per painting; each subfolder contains every photo of that piece (full canvas, install view, detail crops, video stills).

```
inspiration/
  01/   art-01.jpeg
  02/   art-02.jpeg
  ...
  06/   art-06.jpeg  art-07.jpeg     <- multi-photo painting
  ...
```

23 paintings total · 32 photos. Use **Add file → Upload files** at the appropriate `inspiration/NN/` URL to add more photos of an existing piece, or create a new numbered folder for a new piece.
